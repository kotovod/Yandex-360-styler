# АСИТ Дейли - Frontend

Progressive Web App для учёта АСИТ-терапии препаратом Сталораль.

## Технологии

- React 18 + TypeScript
- Vite
- Tailwind CSS
- PWA (Service Worker + Web Push API)
- Zustand для состояния
- Axios для API
- date-fns для работы с датами
- lucide-react для иконок

## Установка

```bash
npm install
```

## Запуск разработки

```bash
npm run dev
```

## Сборка для продакшена

```bash
npm run build
```

## Предварительный просмотр сборки

```bash
npm run preview
```

## Настройка

Скопируйте `.env.example` в `.env` и настройте переменные окружения:

```bash
cp .env.example .env
```

## Функции

- ✅ Автоматический расчёт дозы по дням терапии
- ✅ Отметка приёма и пропуска
- ✅ Календарь с историей приёмов
- ✅ Отслеживание побочных эффектов
- ✅ Push-уведомления
- ✅ Экспорт данных для врача
- ✅ Офлайн-режим
- ✅ Установка на домашний экран
