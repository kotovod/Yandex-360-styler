# Создание иконок для PWA

Для работы PWA необходимы иконки. Вы можете:

## Вариант 1: Использовать готовую иконку

Поместите файлы `pwa-192x192.png` и `pwa-512x512.png` в папку `frontend/public/`

## Вариант 2: Создать из SVG/PNG

Если у вас есть иконка, используйте онлайн-генератор:
- https://realfavicongenerator.net/
- https://www.pwabuilder.com/imageGenerator

## Вариант 3: Использовать placeholder

Для тестирования можно создать простые цветные квадраты:

```bash
# На Mac с ImageMagick
convert -size 192x192 xc:#6366f1 frontend/public/pwa-192x192.png
convert -size 512x512 xc:#6366f1 frontend/public/pwa-512x512.png

# Или онлайн:
# https://placeholder.com/
# Скачайте изображения 192x192 и 512x512
```

## Рекомендации для иконки

- Простой, узнаваемый дизайн
- Хорошо читается в маленьком размере
- Цветовая схема: голубой/фиолетовый (как в приложении)
- Можно использовать символ таблетки/капли
- Формат: PNG с прозрачным фоном
