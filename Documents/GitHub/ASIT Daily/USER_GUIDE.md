# Инструкция по использованию АСИТ Дейли

## Первый запуск

### 1. Регистрация

При первом открытии приложения вам нужно зарегистрироваться:
- Введите ваше имя
- Введите email
- Придумайте пароль (минимум 6 символов)

### 2. Настройка терапии

После регистрации перейдите в раздел "Настройки" (нижнее меню):

1. **Дата начала терапии** - укажите дату, когда вы начали курс АСИТ
2. **Поддерживающая доза** - выберите количество нажатий для поддерживающей фазы (рекомендуется 4-8)
3. **Время напоминания** - установите время, когда вы хотите получать напоминания о приёме

Нажмите "Сохранить"

### 3. Включение уведомлений

Для получения push-уведомлений:
1. В разделе "Настройки" найдите блок "Уведомления"
2. Нажмите "Включить"
3. Разрешите уведомления в браузере

## Ежедневное использование

### Отметка приёма препарата

1. Откройте приложение (вкладка "Главная")
2. Проверьте указанное количество нажатий
3. Примите препарат согласно инструкции
4. Нажмите кнопку "Принял препарат"
5. При необходимости добавьте заметку

### Отслеживание побочных эффектов

Если вы заметили побочный эффект:

1. На главном экране нажмите "Побочный эффект"
2. Выберите тип реакции (зуд, отёк, покраснение, другое)
3. Укажите степень тяжести (лёгкий, средний, тяжёлый)
4. Добавьте описание (необязательно)
5. Нажмите "Сохранить"

⚠️ **Важно**: При тяжёлых побочных эффектах немедленно обратитесь к врачу!

### Просмотр истории

1. Перейдите во вкладку "История"
2. Вы увидите список всех приёмов с датами
3. Зелёная галочка ✓ - препарат принят
4. Красный крестик ✗ - приём пропущен

## Фазы терапии

Приложение автоматически управляет дозировкой:

### Фаза 1: Начальная (Дни 1-10)
- Используйте флакон с **голубой крышкой** (10 ИР/мл)
- Доза увеличивается от 1 до 10 нажатий

### Фаза 2: Переход (Дни 11-18)
- Переходите на флакон с **фиолетовой крышкой** (300 ИР/мл)
- Доза увеличивается от 1 до 8 нажатий

### Фаза 3: Поддерживающая (День 19+)
- Продолжайте использовать флакон с **фиолетовой крышкой** (300 ИР/мл)
- Постоянная доза (та, что вы указали в настройках)

## Экспорт для врача

Перед визитом к врачу вы можете экспортировать историю:

1. Откройте раздел "Настройки"
2. Найдите блок "Экспорт данных"
3. Нажмите "Экспорт в PDF"
4. Файл автоматически скачается на ваше устройство

PDF-отчёт содержит:
- Информацию о терапии
- Статистику приверженности
- Историю всех приёмов
- Записи о побочных эффектах

## Установка приложения на телефон

### iOS (Safari)

1. Откройте приложение в Safari
2. Нажмите кнопку "Поделиться" (квадрат со стрелкой)
3. Выберите "На экран Домой"
4. Нажмите "Добавить"

### Android (Chrome)

1. Откройте приложение в Chrome
2. Нажмите три точки (⋮) в правом верхнем углу
3. Выберите "Установить приложение" или "Добавить на главный экран"
4. Нажмите "Установить"

После установки приложение будет работать как обычное приложение с иконкой на главном экране.

## Рекомендации

✅ **Принимайте препарат в одно и то же время каждый день**
✅ **Не пропускайте приёмы - это снижает эффективность терапии**
✅ **Отмечайте приём сразу после приёма препарата**
✅ **Записывайте все побочные эффекты, даже незначительные**
✅ **Регулярно показывайте историю врачу**

## Техническая поддержка

Если у вас возникли проблемы:
- Проверьте подключение к интернету
- Обновите приложение (перезагрузите страницу)
- Очистите кэш браузера
- Попробуйте другой браузер

## Безопасность

- Ваши данные хранятся в облаке и защищены
- Используйте надёжный пароль
- Не передавайте данные для входа третьим лицам
- Выходите из аккаунта на общих устройствах
