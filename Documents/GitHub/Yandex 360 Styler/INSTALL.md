# Инструкция по установке

## Установка расширения

### Для Chrome/Edge:

1. **Откройте страницу расширений:**
   - Chrome: `chrome://extensions/`
   - Edge: `edge://extensions/`

2. **Включите режим разработчика:**
   - Переключите тумблер "Режим разработчика" в правом верхнем углу

3. **Загрузите расширение:**
   - Нажмите "Загрузить распакованное расширение"
   - Выберите папку с расширением

4. **Проверьте установку:**
   - В списке расширений должно появиться "YaMail Custom UI"
   - Иконка расширения появится в панели инструментов

## Использование

1. **Откройте Яндекс.Почту:**
   - Перейдите на `mail.yandex.ru`

2. **Откройте настройки расширения:**
   - Нажмите на иконку расширения в панели инструментов

3. **Настройте кастомизацию:**
   - **Логотип:** Загрузите изображение (PNG, JPG, SVG)
   - **Цвет кнопок:** Выберите цвет для кнопок интерфейса
   - **Фон меню:** Выберите цвет фона левого меню

4. **Примените изменения:**
   - Нажмите кнопку "Применить"
   - Изменения вступят в силу немедленно

## Сброс настроек

- Нажмите кнопку "Сбросить" в интерфейсе расширения
- Или удалите расширение и установите заново

## Устранение проблем

**Расширение не работает:**
- Убедитесь, что вы находитесь на сайте mail.yandex.ru
- Обновите страницу после применения настроек
- Проверьте, что расширение включено в chrome://extensions/

**Изменения не применяются:**
- Попробуйте обновить страницу
- Проверьте консоль браузера на наличие ошибок
- Убедитесь, что файл логотипа не поврежден
